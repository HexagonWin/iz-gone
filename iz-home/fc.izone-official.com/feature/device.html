推奨環境について

対応デバイス
PC・スマートフォンのみ
※当サイトはガラケー、ガラホ（テンキーの付いたスマートフォン）、らくらくスマートフォンは非対応となります。
※タブレット端末は表示方式によって正常にページが表示されない場合がございます。

対応OS・ブラウザ
●対応OS
・iOSの場合、iOS10.0以上
・Androidの場合、5以上

●対応ブラウザ
・iOSの場合、Safariの最新版
・Androidの場合、Chromeの最新版またはAndroidブラウザ
・Windowsの場合 IE11、Microsoft Edge、Chromeの最新版
・Mac OS Xの場合、SafariまたはChromeの最新版

JavaScript
本サイトはJavaScriptを使用しています。お使いのブラウザでJavaScriptの設定が無効になっていますと動作いたしませんので、有効にしてください。

Cookie
本サイトはCookieを使用しています。お使いのブラウザでCookieの設定が無効になっていますと動作いたしませんので、有効にしてください。

Recommended Environments

Supported devices
PC or smartphones only
※This site does not support tablet devices, Galaho (smartphones using Tenki), or Rakuraku smartphones.

Browser and OS support
● OS support
・For iOS, versions 10.0 and above
・For Android, versions 5 and above

● Supported browsers
・For iOS, latest version of Safari
・For Android, latest version of Chrome or Android browser.
・Windows, Internet Explorer 11, Microsoft Edge or latest version of Chrome
・Mac OS X, Safari or latest version of Chrome

JavaScript
This site uses JavaScript. If the browser you are using does not have JavaScript enabled, the site will not function correctly. Please enable it in your browser settings.

Cookie
This site uses Cookie. If the browser you are using does not have Cookie enabled, the site will not function correctly. Please enable them in your browser settings.
