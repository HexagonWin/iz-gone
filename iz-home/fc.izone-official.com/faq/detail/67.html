よくあるご質問・お問い合わせ

Q.
로그인 ID를 잊어버렸어요.
A.
로그인 ID는 등록시 메일주소 혹은 회원번호 입니다.



확인할 수 없는 경우에는 아래 필요사항을 기재하신 후 문의해 주세요.

※정확한 정보를 입력하지 않으시면, 답변이 늦어지거나 답변을 드리지 못 할 수 있습니다.



・성명(한자/후리가나)

・생년월일(서력)

・전화번호(자택/휴대전화)

・주소

・등록 시 메일주소



【월정액 플랜으로 등록하신 경우】

아래 정보를 기재하신 후 문의해 주세요.

=====================

・등록 닉네임

・등록 도도부현

・생년월일(8자리 서력 표기)

・결제 정보

=====================

▽docomo：sp모드 결제  결제 번호 확인 방법

---------------------------------------------

1. d메뉴 톱→고객 서포

2. 요금 확인【요금 확인 목록 보기】을 선택

3. 상품 등의 구입 이력 확인【sp모드 결제・도코모 휴대폰 지불】

4. 네트워크 비밀번호 입력

5. 【이용 상세】→[계속 과금・이력 목록]

6.  [상세]에서 결제 번호(4자리마다 하이픈으로 나뉘 12자리
숫자)를

확인하신 후 연락해 주십시오.

---------------------------------------------

▽au：간단 결제   계속 과금 ID의 확인 방법

----------------------------------------

1. <au 스마트 패스>에 접속합니다.

※au 스마트 패스 앱이 아니라, web사이트에서 검색해 주십시오.

2. 우선 au ID로 로그인합니다. [로그인]을 클릭해 주십시오.

3. [au ID]에 로그인합니다.

4. 우측 상단의 [au ID] 또는 [au ID 회원 정보]를 클릭합니다.

5. 우측 상단의 [MENU]을 클릭합니다.

6. [계속 이용 서비스 목록]을 클릭합니다.

7. 서비스명을 클릭하면 상세 정보를 참조하실 수 있습니다.

8. 이 화면에 기재된 [계속 과금 ID(8~9자리)]를 알려 주십시오.

----------------------------------------

▽SoftBank：통합 지불

 주문 번호의 확인 방법

----------------------------------------

1. 브라우저에서 Yahoo！Japan에 접속해 주십시오.

2. 페이지 하단에 「MY SoftBank」 표시가 있으므로, 클릭합니다.

3. 로그인 후 ⇒ '이용 요금 확인하기'를 클합니다.

4. '소프트뱅크 통합 지불 이용 이력'을 클릭합니다.

5. '이용 이력'에서 【주문 번호】를 확인하신 후 알려 주십시오.
